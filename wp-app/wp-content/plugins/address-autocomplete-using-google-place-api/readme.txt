Address autocomplete using google place api. Built with the security standards for the most recent WP version.

== Description ==
This plugin allows you to enable google place address autocompletion on any text field  by using it's name, id or class attribute value in admin backend setting.

= Note =
This plugin is ready to use with any theme.

== Installation ==

 Quick Setup


 1) Install plugin through backend of your Wordpress site or via FTP.
 

 Having trouble with installation? Just check our website with free videos on installation and setup.

 From Wordpress Backend (automatic):
 Go to `Plugins -> Add new` -> click on Upload Plugin and upload the File address-autocomplete-using-google-place-api.zip,
 or by Search Plugins, type in:`address-autocomplete-using-google-place-api` and click on ìnstall`

 Manual Setup:
 1. Upload `address-autocomplete-using-google-place-api` Folder from unzip address-autocomplete-using-google-place-api.zip to the `/wp-content/plugins/` directory
 2. Activate the plugin through the 'Plugins' menu in WordPress
